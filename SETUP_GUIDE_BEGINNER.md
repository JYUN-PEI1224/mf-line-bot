# 🔰 初心者向け：マネーフォワード勤怠 LINE Bot セットアップ完全ガイド

このガイドでは、プログラミングやサーバーの知識が少ない方でも、LINEからマネーフォワード勤怠の打刻ができるシステムを作れるように、手順を一つずつ丁寧に解説します。

---

## 📝 全体の流れ（3つのステップ）

1. **LINE公式アカウントを作る**（LINEの設定）
2. **システムを動かす場所（サーバー）を用意する**（Renderという無料サービスを使います）
3. **LINEの画面にメニューボタンを作る**（仕上げ）

所要時間の目安：約30分〜1時間

---

## Step 1: LINE公式アカウントを作る

まずは、あなたが打刻をするための「LINE Bot（ロボット）」を作成します。

### 1-1. LINE Developersに登録する
1. パソコンで [LINE Developers](https://developers.line.biz/ja/) にアクセスします。
2. 右上の「ログイン」をクリックし、普段使っているLINEアカウントでログインします。
3. 初めての場合は、開発者としての名前（自分の名前でOK）とメールアドレスを登録します。

### 1-2. プロバイダー（グループ）を作る
1. 「新規プロバイダー作成」をクリックします。
2. プロバイダー名に「勤怠システム」など好きな名前を入力して作成します。

### 1-3. LINE Botのチャネルを作る
1. 作成したプロバイダーの画面で、「**Messaging API**」を選択します。
2. 以下の情報を入力します：
   - **会社・事業者の所在国・地域**: 日本
   - **チャネル名**: 「勤怠Bot」など、LINE上で表示される名前
   - **チャネル説明**: 「勤怠打刻用です」など
   - **大業種・小業種**: 適当なもの（「IT・コンピュータ」など）を選択
3. 利用規約にチェックを入れて「作成」をクリックします。

### 1-4. 2つの「秘密の鍵」をメモする（超重要！）
Botを動かすためのパスワードのようなものを2つ取得します。メモ帳などにコピーしておいてください。

**① チャネルシークレット**
- 「チャネル基本設定」タブを下にスクロールすると「チャネルシークレット」という項目があります。
- この文字列（英語と数字の羅列）をコピーしてメモします。

**② チャネルアクセストークン**
- 「Messaging API設定」タブを開き、一番下までスクロールします。
- 「チャネルアクセストークン（ロングターム）」の「発行」ボタンを押します。
- 出てきた非常に長い文字列をコピーしてメモします。

---

## Step 2: システムを動かすサーバーを用意する

作成したLINE Botを24時間動かすための「サーバー」を用意します。ここでは初心者でも簡単に無料で使える「**Render（レンダー）**」というサービスを使います。

### 2-1. Renderに登録する
1. [Renderの公式サイト](https://render.com/) にアクセスします。
2. 「GET STARTED FOR FREE」をクリックし、メールアドレスとパスワードでアカウントを作成します。

### 2-2. Webサービスを作成する
1. Renderのダッシュボード（管理画面）で「**New +**」ボタンを押し、「**Web Service**」を選択します。
2. 「Build and deploy from a Git repository」を選択し、「Next」を押します。
3. 「Public Git repository」の入力欄に以下のURLを貼り付けます：
   *(※ここに今回作成したプログラムをGitHub等にアップロードしたURLを入力します。今回は手動アップロードの手順を解説します)*

> 💡 **初心者向けの簡単アップロード方法**
> プログラムのファイルをRenderに直接アップロードすることはできないため、無料のGitHubアカウントを作成してファイルをアップロードする必要があります。
>
> 1. [GitHub](https://github.com/) で無料アカウントを作成します。
> 2. 「New repository」から新しいリポジトリ（ファイルの保管庫）を作成します（名前は「mf-line-bot」など）。
> 3. **このチャット画面に添付されている「mf-line-bot.zip」をパソコンにダウンロードして解凍します。**
> 4. 解凍したフォルダの中にあるファイル（`app.py`や`requirements.txt`など）をすべて選択し、GitHubの画面にドラッグ＆ドロップしてアップロードします。
> 5. 画面下部の「Commit changes」という緑色のボタンを押して保存します。
> 6. Renderの画面に戻り、「Connect a repository」でGitHubを連携させ、今作ったリポジトリを選択します。

### 2-3. サーバーの設定を入力する
以下の通りに設定します：
- **Name**: `mf-line-bot`（好きな名前でOK）
- **Region**: `Singapore` または `Oregon`
- **Branch**: `main`
- **Runtime**: `Docker`
- **Instance Type**: `Free`（無料プラン）

### 2-4. 「秘密の鍵」をサーバーに教える（環境変数）
下にスクロールし、「**Environment Variables**（環境変数）」という項目の「Add Environment Variable」を押して、Step 1でメモした鍵を登録します。

1. **1つ目**
   - Key: `LINE_CHANNEL_SECRET`
   - Value: （Step 1でメモしたチャネルシークレットを貼り付け）
2. **2つ目**
   - Key: `LINE_CHANNEL_ACCESS_TOKEN`
   - Value: （Step 1でメモしたチャネルアクセストークンを貼り付け）

入力できたら、一番下の「**Create Web Service**」をクリックします。
画面に文字がたくさん流れますが、5〜10分ほど待って「Live」という緑色のマークが出れば成功です！

### 2-5. サーバーのURLをメモする
画面の左上に `https://mf-line-bot-xxxx.onrender.com` のようなURLが表示されています。これをコピーしてメモします。

---

## Step 3: LINEとサーバーをつなぐ（仕上げ）

### 3-1. Webhook（連携）の設定
LINEの画面でボタンが押されたとき、先ほど作ったサーバーに「ボタンが押されたよ！」と伝える設定をします。

1. [LINE Developers](https://developers.line.biz/ja/) に戻り、「Messaging API設定」タブを開きます。
2. 「Webhook URL」の「編集」を押し、Step 2-5でメモしたURLの末尾に `/webhook` を付けたものを入力します。
   - 例：`https://mf-line-bot-xxxx.onrender.com/webhook`
3. 「更新」を押し、その下の「Webhookの利用」をオンにします。
4. 「検証」ボタンを押して「成功」と出れば連携完了です！

### 3-2. 自動返信をオフにする
1. 同じ画面の「LINE公式アカウント機能」にある「応答メッセージ」の「編集」を押します。
2. 別タブで設定画面が開くので、「応答メッセージ」を**オフ**、「Webhook」を**オン**にします。

### 3-3. メニューボタン（リッチメニュー）を作る
スマホのLINE画面の下に表示されるボタンを作ります。

パソコンの「コマンドプロンプト（Windows）」または「ターミナル（Mac）」を開き、以下のコマンドを1行ずつコピーして貼り付け、Enterキーを押します。

```bash
# 1. 必要なツールをインストール
pip install requests Pillow

# 2. あなたのLINEの鍵を設定（"あなたのトークン"の部分をStep 1-4の②に置き換えてください）
# Windowsの場合：
set LINE_CHANNEL_ACCESS_TOKEN="あなたのトークン"
# Macの場合：
export LINE_CHANNEL_ACCESS_TOKEN="あなたのトークン"

# 3. メニュー作成プログラムを実行
python setup_richmenu.py
```
「✅ リッチメニューのセットアップが完了しました！」と表示されれば成功です。

---

## 🎉 完成！実際に使ってみよう

### 1. Botを友だち追加する
LINE Developersの「Messaging API設定」タブにあるQRコードをスマホのLINEで読み取り、Botを友だち追加します。

### 2. マネーフォワードの設定をする
1. Botのトーク画面を開き、「**設定**」とメッセージを送信します。
2. Botが返信してくるので、案内に従って以下の3つを送信します：
   - **会社ID**（マネーフォワードログインURLの `https://attendance.moneyforward.com/` の後の部分、または管理者から指定されたID）
   - **メールアドレス**
   - **パスワード**

### 3. 打刻する
画面下のメニューから「出勤」や「退勤」ボタンを押すだけで、マネーフォワードに自動で打刻されます！
「状況確認」を押せば、今日の打刻時間がLINEに返ってきます。

## ❓ よくある質問（トラブルシューティング）

**Q. 「検証」ボタンを押すとエラーになる**
A. Renderのサーバーがまだ起動していない（Liveになっていない）か、Webhook URLの末尾に `/webhook` を付け忘れている可能性があります。

**Q. 「不明な操作です」と返ってくる**
A. LINEの自動応答メッセージがオンになっている可能性があります。Step 3-2の手順を確認してください。

**Q. メニューボタンが表示されない**
A. 一度Botのトーク画面を閉じて、もう一度開いてみてください。それでも出ない場合は、Step 3-3のプログラム実行でエラーが出ていないか確認してください。
