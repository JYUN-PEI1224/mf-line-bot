# コマンドを使わずにLINE管理画面からリッチメニューを作る方法

黒い画面（コマンドプロンプトやターミナル）を使わずに、ブラウザの操作だけでリッチメニュー（LINEの下に出てくるボタン）を作成する方法を解説します。

---

## 1. LINE Official Account Manager にログインする

1. [LINE Official Account Manager](https://manager.line.biz/) にアクセスします。
2. 「LINEアカウントでログイン」または「ビジネスアカウントでログイン」を選び、作成したボットのアカウント管理画面を開きます。

---

## 2. リッチメニューの作成画面を開く

1. 左側のメニューから **「トークルーム管理」** をクリックします。
2. その下にある **「リッチメニュー」** をクリックします。
3. 画面右上の緑色の **「作成」** ボタンをクリックします。

---

## 3. 基本設定を入力する

「表示設定」という画面が出たら、以下のように入力します。

| 項目 | 入力する内容 |
|---|---|
| **タイトル** | `勤怠メニュー`（管理用なので何でもOKです） |
| **表示期間** | 本日の日付 ～ 2030年などの未来の日付（ずっと表示させるため） |
| **メニューバーのテキスト** | `メニュー` |
| **デフォルトの表示** | `表示する` を選択 |

---

## 4. コンテンツ設定（ボタンの配置と動作）を入力する

次に、実際のボタンの動作を設定します。

### ① テンプレートの選択
1. 「テンプレートを選択」をクリックします。
2. **「大（6分割）」** のテンプレート（2段×3列のマス目）を選んで「選択」を押します。

### ② 画像のアップロード
背景画像が必要です。ご自身で作成した画像（横2500px × 縦1686pxなど）をアップロードするか、**「画像を作成」** ボタンを押して、簡易的な背景色とテキスト（出勤、退勤など）を配置して作成することもできます。

### ③ アクションの設定（重要）
ここが一番重要です！各ボタン（A〜F）を押したときに、**Botに特定の文字を送信させる**設定をします。

| ボタン | タイプ | テキスト（一言一句この通りに入力してください） |
|---|---|---|
| **A** (左上) | テキスト | `出勤` |
| **B** (中上) | テキスト | `退勤` |
| **C** (右上) | テキスト | `休憩開始` |
| **D** (左下) | テキスト | `休憩終了` |
| **E** (中下) | テキスト | `状況確認` |
| **F** (右下) | テキスト | `設定` |

※すべて「テキスト」タイプを選び、上記の文字を入力します。

---

## 5. 保存して完了！

一番下にある緑色の **「保存」** ボタンをクリックします。

これで完了です！
お手元のスマホでLINEアプリを開き、作成したBotのトーク画面を開くと、設定したメニューが表示されます。ボタンを押して「出勤」などの文字が自動で送信されれば成功です！
